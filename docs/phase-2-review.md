# 第二阶段 Review：富文本发布预览与推广素材

## 本阶段完成内容

- Obsidian 内富文本发布预览工作台
- 发布准备评分
- 发布前检查清单
- 本地与远程图片预览
- 标题、段落、列表、引用、Callout、代码块和表格渲染
- 54 秒推广概念视频
- 推广视频脚本、旁白、字幕与落地页文案

## 插件 Review

1. 覆盖插件目录中的 `main.js`、`manifest.json`、`styles.css`。
2. 重启 Obsidian，确认版本为 `0.5.0`。
3. 打开一篇包含多种 Markdown 结构的真实长文。
4. 执行：

```text
VaultRelay: 预览当前文章成品
```

请检查：

- 发布准备评分是否容易理解。
- 警告内容是否真实有用。
- 标题、段落、列表和引用的阅读效果。
- 代码块和表格是否正确。
- 公开 OSS 图片是否正常显示。
- Obsidian 本地图片是否能解析并显示。
- 预览工作台是否让你产生“这是最终成品”的感觉。

## 当前限制

- 推广概念视频没有配音和背景音乐。
- 视频中的平台成品包属于产品方向展示，不代表目前全部完成。
- 当前插件只完成平台无关结构解析与本地富文本预览。
