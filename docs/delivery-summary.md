# Obsidian Publish Studio 0.6.0 交付摘要

## 当前可集中验证的产品闭环

```text
Obsidian 长文
→ 结构化解析
→ 发布前检查
→ 富文本成品预览
→ 长文图片卡片
→ 批量导出 PNG 回当前 Vault
```

## 插件命令

```text
Obsidian Publish Studio: 检查当前文章结构
Obsidian Publish Studio: 预览当前文章成品
Obsidian Publish Studio: 生成长文图片卡片
```

> 不迁移 Vault 的 Obsidian 长内容创作者，是否愿意使用并付费购买“发布前检查 + 成品预览 + 图片卡片导出”工作流。
