# Website media assets

Replace these files without editing `site/index.html`:

- `vaultrelay-demo.mp4`: homepage product demonstration video, recommended 16:9 MP4.
- `vaultrelay-demo-poster.png`: video cover image, recommended 1600x900 PNG or WebP.
- `customer-service-wechat.png`: customer-service WeChat QR code displayed in the purchase section.

Keep each filename unchanged so deployments update automatically.
